To work with Angular application:
- go to the folder in which UI project is present (\\Source\Repos\FarmAPI\UI)
- open command prompt (by entering cmd in the taskbar)
- run 'ng new <Angular_Project_Name>' to create to new Angular project
- add Angular routing
- use default stylesheet format
- change directory to the newly created angular project to work
- run 'code .' to open the project in VS Code
- run 'ng serve --open' to run the project and open it in the browser
